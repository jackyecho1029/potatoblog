## 🖼️ 第二阶段：概念可视化

### **目标**
使用 Midjourney 生成初始设计概念，由 Gemini 的研究指导。

### **Midjourney 提示词公式**

```
基础结构：
[主题] + [风格] + [材质/质感] + [色彩板] + [情绪] + [技术参数]

示例：
"奢华睡衣套装，极简斯堪的纳维亚风格，有机竹纤维质感，
柔和鼠尾草绿和奶油黄色调，宁静舒适氛围，
--ar 3:4 --style raw --s 250"
```

### **概念变体策略**
1. **初始生成**: 使用 `/imagine` 创建4个基础概念
2. **精细化**: 使用 `U1-U4` 放大喜欢的版本
3. **变体**: 应用 `V1-V4` 生成替代解释
4. **风格一致性**: 保存最佳结果作为 `--sref` 用于系列连贯性

### **Gemini 的角色**
- 基于趋势研究生成10-15个 Midjourney 提示词
- 提供A/B测试的提示词变体
- 建议特定美学的关键词组合

## 🌸 第三阶段：图案设计

### **目标**
使用 Midjourney 的高级功能创建无缝、可平铺的纺织图案。

![图案设计示例](pattern_examples_1769807840192.png)

### **无缝图案提示词**

```
可平铺图案模板：
"无缝重复图案，[图案类型]，[风格]，[配色方案]，
平面设计，高对比度，--tile --ar 1:1 --s 100"

示例：

1. 几何图案：
"无缝重复图案，极简六边形，包豪斯风格，
海军蓝和金色，平面设计，高对比度，--tile --ar 1:1 --s 100"

2. 花卉图案：
"无缝重复图案，精致樱花，日本木版画风格，
灰紫色和奶油色，水彩质感，--tile --ar 1:1 --s 150"

3. 抽象图案：
"无缝重复图案，有机流动形状，中世纪现代，
陶土色和鼠尾草绿，手绘美学，--tile --ar 1:1 --s 120"
```

### **图案编辑 - Vary Region 功能**
*(替代 Firefly 生成填充)*

1. **选择区域**: 使用 Midjourney 的 `Vary (Region)` 按钮
2. **画笔选择**: 高亮显示要修改的特定图案元素
3. **提示词优化**: 仅描述更改内容（例如："添加小圆点"）
4. **迭代**: 重复直到图案达到理想复杂度

### **图案测试清单**
- [ ] 无缝平铺（用2x2网格测试）
- [ ] 保持色彩一致性
- [ ] 适当缩放（在50%、100%、200%测试）
- [ ] 导出高分辨率PNG（使用 `--quality 2` 放大）

## 👔 第四阶段：系列开发

### **目标**
创建具有一致视觉识别的连贯产品系列。

![睡衣系列展示](sleepwear_collection_1769807852456.png)

### **样式引用工作流程**

```
步骤1：建立基础风格
生成您的主打单品：
"优雅睡袍，流动丝绸质感，深勃艮第配金色刺绣，
奢华精致，工作室摄影，--ar 2:3 --style raw"

步骤2：保存为样式引用
复制图片URL，然后使用：
"配套睡裤，相同美学，--sref [粘贴URL] --ar 2:3"

步骤3：扩展系列
- 睡裙："配套睡裙，--sref [URL] --ar 2:3"
- 拖鞋："协调拖鞋，--sref [URL] --ar 1:1"
- 眼罩："睡眠眼罩，--sref [URL] --ar 3:2"
```

### **系列一致性技巧**
- 使用 `--sref` 在5-10件单品间保持视觉连贯性
- 保持相同的 `--s`（风格化）值（例如 `--s 250`）
- 在产品类别内保持纵横比一致
- 在同一 Midjourney 会话中生成所有单品以确保色彩准确

## ⚙️ 技术规格与参数

### **Midjourney 参数速查表**

```
核心参数：

--ar [比例]        纵横比（例如：睡衣用 --ar 3:4，图案用 --ar 1:1）
--s [0-1000]      风格化（100=写实，250=平衡，750=艺术）
--style raw       更摄影化，较少艺术化解释
--tile            创建无缝重复图案
--sref [URL]      样式引用以保持一致性
--quality 2       更高细节（消耗2倍积分）
--no [元素]       排除不需要的元素（例如：--no text, --no people）

高级参数：
--chaos [0-100]   变化多样性（0=一致，100=狂野）
--weird [0-3000]  实验性美学
--personalize     使用您的美学偏好（需要设置）
```

### **按用例推荐的设置**

| 用例 | 纵横比 | 风格化 | 其他 |
|------|--------|--------|------|
| 睡衣产品 | --ar 2:3 | --s 250 | --style raw |
| 平铺造型 | --ar 4:3 | --s 200 | --style raw |
| 无缝图案 | --ar 1:1 | --s 100 | --tile |
| 情绪板 | --ar 16:9 | --s 300 | - |
| 细节特写 | --ar 1:1 | --s 150 | --quality 2 |

## ✅ 质量控制与导出

### **导出前检查清单**

- [ ] **色彩准确性**: 与Pantone/十六进制代码对比
- [ ] **分辨率**: 放大至最小2048px（使用 `U` 按钮或 `--quality 2`）
- [ ] **图案平铺**: 在Photoshop/Illustrator中测试无缝图案
- [ ] **品牌一致性**: 验证所有单品符合风格指南
- [ ] **文件命名**: 使用描述性名称（例如：`2026_春季_睡袍_鼠尾草绿_V3.png`）

### **导出工作流程**

1. **Midjourney**: 下载全分辨率图片（点击图片 → 在浏览器中打开 → 保存）
2. **放大**（如需要）: 使用 Midjourney 的 `Upscale (2x)` 或 `Upscale (4x)` 按钮
3. **后期处理**（可选）:
   - 在Photoshop中调整亮度/对比度
   - 移除背景（使用Remove.bg或Photoshop的对象选择）
   - 添加样机（使用Smartmockups或Placeit）
