## 📚 关键词

从视频中提取的 5 个重要专业概念：

**1. 命令行界面（CLI - Command Line Interface）**

> **含义：** 一种通过输入文本指令来与计算机交互的方式。相比图形界面，它更直接、更高效，是专业开发者的核心阵地。

**💼 案例：** **Git** 是最成功的 CLI 工具案例，它让全球数千万程序员通过简单的指令就能管理复杂的代码版本，而 Claude Code 正在以同样的方式管理编程逻辑。

---

**2. 代理式 AI（Agentic AI）**

> **含义：** 不仅能生成文本，还能根据目标自主思考、调用外部工具并执行动作的 AI 系统。

**💼 案例：** **Salesforce** 正在利用代理式 AI 自动处理客户电邮并更新数据库信息，这种“思考并行动”的模式极大提升了企业自动化水平。

---

**3. 上下文窗口（Context Window）**

> **含义：** AI 在一次处理中能够“记住”和“理解”的信息总量。窗口越大，AI 理解复杂长文本的能力就越强。

**💼 案例：** **Anthropic** 凭借超长的上下文窗口处理能力，让法律从业者能一次性投喂整本合同法案，并从中精准找出逻辑漏洞。

---

**4. 令牌成本（Token Cost）**

> **含义：** 使用大语言模型时的计费单位。由于 Claude Code 调用频繁，管理好 Token 消耗是高效使用的关键。

**💼 案例：** 许多科技初创公司通过优化代码提示词（Prompt），将每月的 AI 调用成本降低了 40%，实现了成本与性能的平衡。

---

**5. 权限控制（Permission Handling）**

> **含义：** 在自动化执行过程中，系统向用户请求确认的关键环节，用于防止非预期的误操作。

**💼 案例：** **AWS** 在其云管理控制台中设置了严格的权限校验，确保任何可能导致服务中断的操作都必须经过多重认证，这与 Claude Code 的安全哲学不谋而合。
